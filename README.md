# 親子活動搜尋器

## 專案目的
幫助安排與 12 歲男孩的週末與連假活動，透過 AI 搜尋台北及近郊
適合孩子體驗、培養品味與視野的活動資訊。

## 核心設計原則
- 最小摩擦：打開即用，不需登入或安裝
- AI 篩選：用 Gemini API 搜尋並整理活動
- 一鍵存入 Notion：與現有 PKM 系統整合

## 技術架構
- 前端：單一 HTML 檔案，無後端
- AI：Gemini 2.0 Flash API（含 Google Search 工具）
- 部署：GitHub Pages
- API Key：使用者手動輸入，不寫入程式碼

## 搜尋條件預設值
- 孩子年齡：12 歲
- 地區：台北市區、新北／基隆、宜蘭／桃園
- 搜尋週數：未來 4 週
